# LeetCode 刷题计划
## 来源说明
本刷题计划源自 **灵茶山艾府** 的 LeetCode 社区分享帖
原帖链接：https://leetcode.cn/circle/discuss/RvFUtj/

## 通用刷题规则
1.  除特殊说明外，优先完成**难度分 ≤ 1700** 的题目
2.  `00_编程入门` 章节包含两道数据库题目，可直接跳过
3.  `02_二分算法` 章节的「二分答案」类题目为选做，若觉得困难可跳过
4.  `03_数据结构` 是「新手村常用工具」，核心内容为枚举技巧、前缀和、栈、队列、堆
5.  第 04-06 章节侧重递归思想的理解，为后续动态规划学习打基础
